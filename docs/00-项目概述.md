# 项目概述

## 背景

一群完全没有开发经验与业务领域专业背景的人，需要利用AI编码构建一套系统。

### 系统构成
- **Server** - 云端服务
- **App** - 移动应用
- **Web** - 管理后台

### 业务领域
- 聚焦于考勤业务
- 业务规则仿照海康云眸

## 目标

在背景条件下尽可能发挥AI编码的优势，创造尽可能优质的demo，以生产能力为目标。

## 团队能力

- 2人团队
- 可以熟练使用 Trae IDE、智能体、Skill 等AI编码能力
- 无传统开发经验
- 无业务领域专业背景

## 约束条件

- 主要使用 Trae IDE 作为开发工具
- Server 暂时使用 Docker 运行
- 技术栈无特定偏好，由讨论确定

## 核心诉求

1. 探索AI编码项目的组织方式
2. 建立人+AI的协作方法论
3. 最大化发挥AI编码能力
4. 考勤系统只是载体，核心是方法论的探索
