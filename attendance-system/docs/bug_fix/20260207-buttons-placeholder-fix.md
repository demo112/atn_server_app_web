# 20260207-buttons-placeholder-fix.md

## 问题描述
- **现象**：Web 端多个页面的 Toolbar 按钮（如“更换部门”、“导入”、“导出”）点击无反应。
- **复现步骤**：进入人员管理页面，点击上述按钮。
- **影响范围**：`PersonnelDashboard.tsx` 及潜在的其他页面。

## 设计锚定
- **所属规格**：SW62
- **原设计意图**：功能尚未开发，UI 仅为占位符。
- **当前偏离**：用户点击无反馈，体验不佳。

## 根因分析
- **直接原因**：按钮缺少 `onClick` 事件处理。
- **根本原因**：功能迭代过程中，UI 先行，逻辑未跟上。
- **相关代码**：`PersonnelDashboard.tsx`

## 修复方案
- **修复思路**：为所有未实现按钮添加 `toast.info('功能暂未开放')`。
- **改动文件**：`PersonnelDashboard.tsx`
- **关联组件**：无其他组件发现类似问题（经搜索排查）。
