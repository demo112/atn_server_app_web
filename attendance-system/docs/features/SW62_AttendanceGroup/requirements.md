# 需求规格说明书：考勤组管理 (SW62_AttendanceGroup)

## 1. 概述
将参考代码 (`incoming/web/attendance_group`) 中的考勤组管理功能迁移至主 Web 项目中。
实现考勤组的增删改查、规则配置、人员排班等功能的前端界面。

## 2. User Stories

### 2.1 考勤组列表
- **As a** 管理员
- **I want** 查看考勤组列表
- **So that** 我可以了解当前所有考勤组的状态（人数、班次、有效期等）
- **AC**:
  - 显示考勤组名称、人数、班次名称、有效期、状态
  - 支持按名称搜索
  - 支持刷新列表
  - 支持分页（UI展示）

### 2.2 新增/编辑考勤组
- **As a** 管理员
- **I want** 新增或编辑考勤组
- **So that** 我可以配置考勤规则
- **AC**:
  - **基本配置**: 名称、人员选择、有效期
  - **打卡配置**: 打卡方式（设备/App）、设备选择
  - 支持模态框选择人员和设备

## 3. 约束条件
- 界面风格需尽可能还原参考代码
- 使用现有的 Tailwind CSS 架构
- 暂时使用 Mock 数据，预留 API 接口位置
